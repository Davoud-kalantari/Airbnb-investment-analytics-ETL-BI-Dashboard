Browser BI Dashboard

Airbnb Investment Analytics ETL & BI Dashboard

This browser dashboard reconstructs the Final Dashboard / Step 4 decision logic from the Tableau workflow using public-safe processed data only. It follows the same business path: occupancy-led screening, best-city selection, best house-type ranking, and budget feasibility analysis.
